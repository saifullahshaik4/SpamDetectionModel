## How to Run

1. Clone this repository.
2. Open `notebook/SpamEmailDetection.ipynb` in Google Colab or Jupyter.
3. Ensure the following libraries are installed:
   - `numpy`
   - `pandas`
   - `scikit-learn`
   - `matplotlib`
4. Run all cells to train and evaluate the models.
